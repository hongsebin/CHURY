FAQ

자주 묻는 질문에 대한 답변입니다.

닉네임은 어떻게 바꾸나요?
닉네임 변경 경로 및 방법은 다음과 같습니다.

▶ CHURY 웹페이지 (http://127.0.0.1:8000/) → 로그인 후 [내 정보] → [개인정보 관리] → [닉네임 변경]

※ 다른 회원님이 사용 중인 닉네임은 사용하실 수 없습니다.
비밀번호는 어떻게 바꾸나요?
비밀번호 변경 경로 및 방법은 다음과 같습니다.

▶ CHURY 웹페이지 (http://127.0.0.1:8000/) → 로그인 후 [내 정보] → [개인정보 관리] → [비밀번호 변경]
▶ 새 비밀번호와 새 비밀번호 확인을 입력하시고 [변경] 버튼을 누르면 적용됩니다.
비밀번호를 잊어버렸습니다. 어떻게 하나요?
비밀번호를 분실하셨을 시에는 CHURY 홈페이지에서 아래 절차를 통해 찾기 절차를 진행할 수 있습니다.

1. CHURY 로그인 화면 내 '비밀번호 찾기'를 클릭해주세요.
2. 회원가입 시 사용한 이메일을 입력하세요.
3. 해당 이메일로 비밀번호 찾기 링크가 담긴 메일을 보내드립니다.
4. 발송된 메일을 확인하시고, 링크를 클릭해서 새로운 비밀번호를 만들 수 있습니다.

※ 위 방법으로 인증 불가능한 분께서는 [추정 아이디 / 닉네임 ] 등 필요사항을 기재하여 고객센터 이메일(project.chury@gmail.com)로 문의해 주시기 바랍니다.
소셜로그인이란 무엇인가요?
소셜로그인이란 SNS계정(카카오, 네이버)을 연동하여 간편하게 로그인하는 서비스입니다.
로그인/회원가입 화면에서 [카카오], [네이버] 버튼을 통해 이용 가능합니다.
SNS 계정으로 가입했는데, ID와 PW를 잊어버렸어요.
SNS 계정으로 최초 가입하셨을 경우, 아이디와 패스워드 입력 없이 해당 소셜 로그인 버튼을 통해 로그인이 가능합니다.
SNS 계정의 ID(혹은 이메일)와 PW를 잊어버리셨을 경우, 해당 SNS 계정 플랫폼(카카오, 네이버)의 계정, 또는 비밀번호 찾기를 이용해주세요.
